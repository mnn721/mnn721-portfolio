# Security Assessment

## Overview

This project focused on assessing an organization's security posture by identifying vulnerabilities and evaluating potential risks to systems and data. The assessment included analyzing existing security practices and recommending improvements based on cybersecurity best practices.

## Objective

* Identify security vulnerabilities.
* Evaluate the organization's current security posture.
* Recommend security improvements.
* Document findings in a professional assessment report.

## Skills Demonstrated

* Vulnerability Assessment
* Risk Assessment
* Security Analysis
* Threat Identification
* Access Control
* Security Documentation

## Security Issues Identified

* Employees sharing passwords
* Default administrator credentials
* Missing firewall rules
* Lack of multifactor authentication

## Recommendations

* Require unique passwords for every employee.
* Replace all default administrator passwords.
* Configure firewall rules to filter unauthorized traffic.
* Implement multifactor authentication for all users.

## Files Included

* Security_Assessment_Report.pdf

## Key Takeaways

This project improved my ability to evaluate security risks, identify common vulnerabilities, and recommend practical solutions to strengthen an organization's security posture. It also reinforced the importance of following cybersecurity best practices to reduce organizational risk.
