# Security Audit

## Overview

This project involved conducting a security audit for Botium Toys by reviewing its security controls and compliance checklist. The objective was to determine whether existing controls were sufficient, identify areas that required improvement, and provide recommendations to strengthen the organization's security posture and regulatory compliance.

## Objectives

* Evaluate existing security controls.
* Review compliance with industry security best practices.
* Identify security gaps and potential risks.
* Recommend improvements to reduce organizational risk.

## Skills Demonstrated

* Security Auditing
* Risk Assessment
* Compliance Assessment
* Security Controls Evaluation
* Security Documentation
* Technical Report Writing

## Controls Reviewed

* Least Privilege
* Password Policies
* Firewall Configuration
* Intrusion Detection Systems (IDS)
* Antivirus Protection
* Encryption
* Disaster Recovery Planning
* Password Management
* Physical Security Controls
* CCTV Surveillance

## Compliance Frameworks Reviewed

* PCI DSS
* GDPR
* SOC 1 & SOC 2

## Key Recommendations

* Strengthen password management practices.
* Implement encryption for sensitive information.
* Develop a disaster recovery plan.
* Improve access control through least privilege.
* Enhance compliance with PCI DSS, GDPR, and SOC requirements.
