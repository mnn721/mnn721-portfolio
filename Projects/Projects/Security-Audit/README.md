# Security Audit

## Overview

This project involved performing a security audit of an organization's existing security controls to evaluate compliance with security best practices. The goal was to identify weaknesses, assess potential risks, and recommend improvements to strengthen the organization's overall security posture.

## Objective

* Review the organization's current security controls.
* Identify areas that do not meet security standards.
* Recommend improvements to reduce organizational risk.
* Document findings in a professional security audit report.

## Skills Demonstrated

* Security Auditing
* Risk Assessment
* Security Controls
* Access Management
* Password Policies
* Firewall Security
* Multifactor Authentication (MFA)

## Tools & Concepts

* Security Audit Checklist
* Risk Analysis
* Compliance Review
* Principle of Least Privilege

## Key Recommendations

* Implement stronger password policies.
* Replace default administrator credentials.
* Configure firewall rules to restrict unauthorized traffic.
* Enable multifactor authentication (MFA).
* Improve access control procedures.

## Files Included

* Security_Audit_Report.pdf

## Key Takeaways

This project strengthened my understanding of how to evaluate organizational security controls and identify vulnerabilities. It also reinforced the importance of documenting findings clearly and recommending practical security improvements.
