# Linux File Permissions

## Overview

This project focused on managing file and directory permissions in a Linux environment. I reviewed existing permissions, identified unauthorized access, and used Linux commands to update permissions according to organizational security policies.

## Skills Demonstrated

- Linux
- File Permissions
- Access Control
- Principle of Least Privilege
- chmod
- ls -la

## Commands Used

```bash
ls -la
chmod o-w project_k.txt
chmod u=rw,g=r,o= .project_x.txt
chmod g-x drafts
```
